# 启动提示词示例

## 启动一轮选择

```text
我想写一个扮猪吃虎爽文，给我几个选项，我选完后你直接写一章。前期羞辱要写足，后面打脸要狠。
```

```text
帮我生成爽文，只问一轮问题，选完直接写 2000 字左右。正文后问我要不要生成漫画风 AI 短剧。
```

## 直接生成正文

```text
不要问我，直接生成一章都市扮猪吃虎爽文，打脸要分两层。
```

```text
你自己定，直接写 2000 字左右，前面要憋屈，后面要爽。
```

## 视频转化

```text
生成视频。
```

```text
把这一章转成漫画风 AI 短剧分镜。
```

```text
给我可灵/即梦/Sora 可用的视频提示词。
```

```text
把正文改成 60 秒竖屏漫画风短剧脚本。
```

## 修改生成结果

```text
这章不够爽，把反派写得更嚣张，打脸更狠。
```

```text
前期羞辱不够深，重写时增加身份羞辱、公开羞辱和底线羞辱。
```

```text
后面打脸太快了，改成事实打脸 + 权威打脸 + 靠山反向打脸。
```
