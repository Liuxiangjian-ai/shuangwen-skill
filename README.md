## 这是什么？

这是一个面向中文网文语境的爽文生成 Skill。

它专门服务于“扮猪吃虎、逆袭打脸、隐藏身份、歪嘴龙王式反转”这类高情绪回报内容。

用户只需要选择 5 个选项，Skill 就会直接生成一章约 2000 字的爽文正文。正文完成后，用户还可以选择是否继续转成漫画风 AI 短剧分镜和视频生成提示词。

> 把“歪嘴龙王式爽感”结构化、模板化、视频化。

---

## 核心特点

- **一轮选择，直接生成**：用户无需填写复杂设定，只需选择 5 个选项。
- **高爽感结构**：强化“深度羞辱 → 反派加码 → 分层打脸 → 章末钩子”。
- **适配中文爽文语境**：支持都市、医学神医、赘婿、玄幻废柴流、职场逆袭等题材。
- **正文 + 短剧双模式**：先生成爽文正文，再按需转为漫画风 AI 短剧分镜。
- **适合 GPTs / Prompt / Agent**：可以作为自定义 GPT、AI 写作助手或短剧生成工作流的基础 Skill。

## 快速开始

当用户说：

```text
我想写一个扮猪吃虎爽文。
```

Skill 会回复一轮选择：

```text
请按“1A，2B，3C，4D，5A”的格式回复即可：

1. 题材类型
A. 都市扮猪吃虎
B. 医学神医
C. 赘婿逆袭
D. 玄幻废柴流
E. 职场逆袭

2. 主角表面身份
A. 普通打工人
B. 规培医生 / 实习医生
C. 被看不起的赘婿
D. 废柴弟子
E. 底层员工

3. 第一章打脸场景
A. 同学聚会 / 宴会羞辱
B. 急诊抢救 / 专家会诊
C. 家族逼迫 / 岳家羞辱
D. 宗门测试 / 擂台考核
E. 公司会议 / 项目危机

4. 主角反转方式
A. 一个电话叫来大人物
B. 一眼看穿关键问题
C. 拿出隐藏证据
D. 随手完成别人做不到的事
E. 被高层当场认出真实身份

5. 文风与爽点密度
A. 番茄网文风，中爽
B. 短剧爽文风，极爽
C. 传统男频风，稳爽
D. 女频逆袭风，情绪爽
E. 自定义
```

用户回复：

```text
1B，2B，3B，4B，5A
```

Skill 将直接生成一章约 2000 字的医学神医爽文。

---

## 工作流

```text
用户选择 5 个选项
→ 生成约 2000 字爽文正文
→ 强化羞辱、加码、打脸和钩子
→ 询问是否继续生成视频
→ 如果用户回复“生成视频”
→ 输出漫画风 AI 短剧分镜与视频提示词
```

---

## 爽点结构

本 Skill 默认使用以下爽文链条：

```text
身份羞辱
→ 公开羞辱
→ 底线羞辱
→ 反派连续加码
→ 主角沉默蓄势
→ 事实打脸
→ 权威打脸
→ 围观者态度反转
→ 章末更大钩子
```

---

## 视频化能力

正文生成完成后，Skill 会询问：

```text
要不要把这一章继续转成漫画风 AI 短剧视频脚本？
回复“生成视频”即可。
```

如果用户选择生成视频，Skill 会输出：

- 视频基础设定
- 角色视觉设定
- 8–12 个短剧分镜
- 每个镜头的画面、台词、情绪和视频提示词
- 统一视频提示词
- 负面提示词

默认视频风格：

```text
漫画风格 AI 短剧，竖屏 9:16，现代国漫漫画风，强冲突，强表情，强反转，适合短视频平台。
```

> 如果当前平台具备视频生成能力，可以进一步调用视频模型；如果不具备，则输出可复制到可灵、即梦、Sora、Runway、Pika 等视频模型中的完整提示词。
